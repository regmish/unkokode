# Privacy Policy

Live Nepali Unicode Input does not collect, transmit, sell, or share personal data.

## What the extension does

The extension transliterates romanized Nepali text into Nepali Unicode inside editable fields in your browser and in the extension popup.

## Data handling

- Transliteration is processed locally in the browser
- No account is required
- No analytics or tracking code is included
- No data is sent to remote servers by the extension

## Local storage

The extension stores user preferences using Chrome extension storage. This may include:

- Whether the extension is enabled
- Default transliteration mode
- Site allow or block preferences
- Custom transliteration mappings
- Typing behavior preferences

These settings are used only to provide the extension's functionality.

## Third parties

The extension does not sell user data to third parties.
The extension does not use user data for advertising.
The extension does not use user data for credit or lending purposes.
